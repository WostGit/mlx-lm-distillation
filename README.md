# PACXAI 10/10 Lean + MLX Artifact

This repository is generated by a single GitHub Actions workflow.

It contains:
- A self-contained Lean proof artifact.
- MLX proof-matched experiments.
- Human-readable reviewer reports.
- SVG graphs, CSV tables, and an HTML report.

Core claim:

If a student artifact is a deterministic function of the audited transcript, every finite recovery attack on the student has an equivalent transcript-level simulator with exactly the same empirical success.

Failure mode:

If the real training pipeline uses omitted route metadata, the theorem does not apply, and a transcript-only audit can falsely pass.
